San Vicente Group Application Installation Guide
===========================================

Required Files (Keep these files in the same directory):
- SanVicenteApp.exe
- project-1.0-SNAPSHOT.jar

Prerequisites:
1. Java 17 or higher
   - Download from: https://adoptium.net/
   - Choose Eclipse Temurin 17 (JDK)
   - During installation, make sure to:
     * Select "Add to PATH" option
     * Choose "Set JAVA_HOME variable"

2. JavaFX SDK 17.0.13
   - Download from: https://gluonhq.com/products/javafx/
   - Choose version 17.0.13, SDK type, Windows x64
   - Extract the downloaded zip file
   - Place the extracted folder at: C:\Program Files\Java\javafx-sdk-17.0.13

Installation Steps:
1. Install Java 17 (run the downloaded installer)
2. Extract JavaFX SDK to the correct location
3. Place SanVicenteApp.exe and project-1.0-SNAPSHOT.jar in the same directory
4. Double-click SanVicenteApp.exe to run the application

Troubleshooting:
- If you get a Java-related error, make sure Java 17 is properly installed
- If you get a JavaFX-related error, verify that JavaFX SDK is in the correct location
- Both the .exe and .jar files must be in the same file directory

For support or questions, contact MCPI Capstone developers technical support.
